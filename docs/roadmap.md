## TECH STACK DECISIONS

### Why Mailerlite (not ConvertKit)
- âœ… Free automation builder (Kit locks behind paywall)
- âœ… 1,000 subscribers free forever
- âœ… Custom fields for personalization
- âœ… Bearer token API (cleaner than Kit)

### Why No Supabase (Yet)
- Mailerlite stores all quiz data via custom fields
- CSV export available anytime
- Only need database when Mailerlite UI becomes limiting (5K+ subscribers)
- Don't build Month 3 infrastructure on Day 1

### Why 9 Questions (Not 11)
- Removed Q4 (platform preference) â€” Not critical for initial match

- 9 questions = <2 min completion (better for mobile)
