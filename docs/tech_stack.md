| Component | Recommendation | Why |
|-----------|----------------|-----|
| Framework | Next.js or Astro | Fast, SEO-friendly, easy deploys |
| Hosting | Vercel or Netlify | Free tier, instant deploys |
| CMS | Markdown files or Notion API | No database needed Day 1 |
| Styling | Tailwind | Fast, consistent |
| Analytics | Plausible or Umami | Privacy-friendly, simple |
| Affiliate tracking | Pretty Links or direct | Start simple |



## REVISED TECH STACK

| Component | Day 1 Tool | Why |
|-----------|------------|-----|
| Landing + Quiz | Carrd + Typeform OR single Next.js page | Fast, no-code option or custom if you want control |
| Email | Mailerlite (forever-free to 1K subs) | Full automation, sequences, drag-drop editor |
| Analytics | GA4 | Free, funnel tracking, good enough |
| A/B Testing | Day 1: Manual (swap headlines). Week 2+: Splitbee or Vercel Edge | Don't over-engineer before traffic |
| Affiliate Tracking | Day 1: UTM params. Month 2+: Rewardful or custom | Start simple |
